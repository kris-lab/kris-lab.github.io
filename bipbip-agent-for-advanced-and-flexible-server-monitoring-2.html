Bipbip agent for advanced and flexible server monitoring

By Chris Stasiak
January 25, 2014

I am pleased to announce that me and my team released very useful and elegant open source tool/agent for server monitoring.

Bipbip is an agent written in ruby to collect server metrics and send them to the CopperEgg RevealMetrics platform. There are plugins for different metrics available and very easy way to extend/write own!

More at Github and RubyGems
